## Core Principles

The system enforces four non-negotiable properties:

1. Determinism  
All outputs are reproducible and machine-verifiable.

2. Integrity  
All files are SHA-256 verifiable with fixed canonical structure.

3. Licensing Clarity  
All systems are distributed with explicit, perpetual licensing instruments.

4. Semantic Stability  
Terminology and meaning are fixed through governance doctrine and schema.
